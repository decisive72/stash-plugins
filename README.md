# decisive72's Stash Plugins

> [!NOTE]
> These plugins are only tested to work on Stash v0.31.0 or above.

## Usage

1. Go to Settings > Plugins
1. Click "Add source" under Available Plugins
1. Enter https://decisive72.github.io/stash-plugins/main/index.yml as the Source URL
1. Select the plugins you'd like to install
